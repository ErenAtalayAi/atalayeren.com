## Kurulum

Projeyi yerel makinenize kurmak için aşağıdaki adımları takip edin:

1. Bu repoyu klonlayın:

    ```bash
    git clone https://github.com/kullanici_adi/proje_adi.git
    ```

2. Proje dizinine gidin:

    ```bash
    cd proje_adi
    ```

3. Gerekli bağımlılıkları yükleyin:

    ```bash
    npm install
    ```
